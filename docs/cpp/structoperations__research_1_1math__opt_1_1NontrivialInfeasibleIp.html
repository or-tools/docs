Detailed Description

The model is 2*x + 2*y + 2*z >= 3 x + y <= 1 y + z <= 1 x + z <= 1 x, y, z in {0, 1}

The IIS has no variable bounds and all other constraints. In particular, the LP relaxation is feasible.
