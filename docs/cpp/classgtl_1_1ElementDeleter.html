Detailed Description

ElementDeleter is an RAII (go/raii) object that deletes the elements in the given container when it goes out of scope. This is similar to std::unique_ptr<> except that a container's elements will be deleted rather than the container itself.

Example: std::vector<MyProto*> tmp_proto; ElementDeleter d(&tmp_proto); if (...) return false; ... return success;

Since C++11, consider using containers of std::unique_ptr instead.
