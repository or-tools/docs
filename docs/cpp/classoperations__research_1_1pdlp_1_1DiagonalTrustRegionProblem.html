Detailed Description

A generic trust region problem of the form: min_{x} (1 / 2) * (x - center_point)^T Q (x - center_point)

  • c^T (x - center_point) s.t. lower_bounds <= (x - center_point) <= upper_bounds ||x - center_point||_W <= radius where |z|_W = sqrt(sum_i w_i z_i^2) is a weighted Euclidean norm. It is assumed that the objective matrix Q is a nonnegative diagonal matrix.
