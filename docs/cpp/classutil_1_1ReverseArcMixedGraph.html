Detailed Description

template<typename NodeIndexType = int32_t, typename ArcIndexType = int32_t>
class util::ReverseArcMixedGraph< NodeIndexType, ArcIndexType >

This graph is a mix between the ReverseArcListGraph and the ReverseArcStaticGraph. It uses less memory:

  • It takes 2 * ArcIndexType * node_capacity()
    • (2 * NodeIndexType + ArcIndexType) * arc_capacity() memory.
  • If the ArcIndexPermutation is needed, then an extra ArcIndexType * arc_capacity() is needed for it.
