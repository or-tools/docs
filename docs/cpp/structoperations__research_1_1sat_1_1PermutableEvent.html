Detailed Description

Computes the minimum sum of the end min and the minimum sum of the end min weighted by y_size_min of all events. It returns false if no permatutation is valid w.r.t. the range of x_start.

Note
this is an exhaustive algorithm, so the number of events should be small, like <= 10. They should also starts in index order.

Optim: If both sums are proven <= to the corresponding threshold, we abort.
