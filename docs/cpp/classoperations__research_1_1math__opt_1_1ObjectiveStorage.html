◆ diff_is_empty()

bool operations_research::math_opt::ObjectiveStorage::diff_is_empty ( const Diff &  diff) const
inline

Functions for working with Diff Returns true if there are no changes (tracked changes before the checkpoint).

Note
when there are new variables with nonzero objective coefficient, the Diff object can be empty (and diff_is_empty will return true), but Update() can return a non-empty ObjectiveUpdatesProto. This behavior MAY CHANGE in the future, so diff_is_empty is true iff Update() returns an empty ObjectiveUpdatesProto (a more intuitive API, harder to implement efficiently).
