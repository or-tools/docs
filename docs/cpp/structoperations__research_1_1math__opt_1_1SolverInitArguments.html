Detailed Description

Arguments passed to Solve() and IncrementalSolver::New() to control the instantiation of the solver.

For convenience, constructors with streamable or/and non-streamable arguments are provided. The non-streamable arguments are cloned so any change made after passing them to this class are ignored.

Usage with streamable arguments:

Solve(model, SOLVER_TYPE_GUROBI, /*solver_args=*‍/{}, SolverInitArguments({ .gurobi = StreamableGurobiInitArguments{ .isv_key = GurobiISVKey{ .name = "some name", .application_name = "some app name", .expiration = -1, .key = "random", } } });

Usage with non-streamable arguments:

NonStreamableGurobiInitArguments gurobi_args; gurobi_args.primary_env = primary_env.get();

Solve(model, SOLVER_TYPE_GUROBI, /*solver_args=*‍/{}, SolverInitArguments(gurobi_args));
