Detailed Description

template<typename IntType, typename T>
class operations_research::glop::StrictITIVector< IntType, T >

Wrapper around an ITIVector to allow (and enforce) creation/resize/assign to use the index type for the size.

Todo:
(user): This should probably move into ITIVector, but note that this version is more strict and does not allow any other size types.
