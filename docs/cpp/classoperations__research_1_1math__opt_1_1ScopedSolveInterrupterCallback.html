Detailed Description

Class implementing RAII for interruption callbacks.

Usage:

SolveInterrupter* const interrupter = ...; { const ScopedSolveInterrupterCallback scoped_intr_cb(interrupter, [](){ ///< Do something when/if interrupter is not nullptr and is triggered. } ... } ///< At this point, the callback will have been removed.

The function RemoveCallbackIfNecessary() can be used to remove the callback before the destruction of this object.
