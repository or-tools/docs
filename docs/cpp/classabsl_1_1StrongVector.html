◆ mutable_get()

template<typename IntType , typename T , typename Alloc = std::allocator<T>>
ParentType * absl::StrongVector< IntType, T, Alloc >::mutable_get ( )
inline

The mutable accessor is useful when using auxiliar methods relying on vector parameters such as JoinUsing(), SplitStringUsing(), etc. Methods relying solely on iterators (e.g. STLDeleteElements) should work just fine without the need for mutable_get(). NB: It should be used only in this case and thus should not be abused to index the underlying vector without the appropriate IntType.
