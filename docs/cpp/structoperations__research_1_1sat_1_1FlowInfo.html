Detailed Description

A single node flow relaxation is a constraint of the form Sum in_flow - Sum out_flow <= demand where each flow variable F_i is in [0, capacity_i] and satisfy F_i <= capacity_i B_i with B_i a Boolean representing the arc usage.

From a generic constraint sum coeff_i X_i <= b, we try to put it in this format. We can first transform all variables to be in [0, max_value].

Then we cover different cases: 1/ A coeff * Boolean, can be easily transformed. 2/ A coeff * Integer in [0, capacity] with Bool => integer == 0 too. 3/ For a general integer, we can always use a Bool == 1 for the arc usage.
