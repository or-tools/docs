circuit(solver, x)

A decomposition of the global constraint circuit, based on some observation of the orbits in an array.

Note: The domain of x must be 0..n-1 (not 1..n) since Java is 0-based.

put the orbit of x[0] in z[0..n-1]

z may not be 0 for i < n-1

when i = n-1 it must be 0
